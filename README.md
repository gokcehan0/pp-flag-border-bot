## 🚀 Getting Started

1. **Clone the repository**
   ```sh
   git clone https://github.com/gokcehan0/pp-flag-border-test-bot.git
   ```
2. **Install dependencies**
   ```sh
   npm install
   ```
3. **Run the bot**
   ```sh
   node index.js
   ```
